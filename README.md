# Games Portal

**Games Portal** is a web application built with **Laravel 12** that allows users to explore, review, and manage video games. Admin users can manage games, categories, and reviews, while regular users can browse games, leave reviews, and manage their profiles. The frontend is built with **Bootstrap 5** for responsive design.

---

## Features

### For All Users
Browse a list of games with images, descriptions, and categories.
View game details and reviews.
Responsive design using Bootstrap 5.

### For Registered Users
Create and edit personal profile.
Leave reviews for games.

### For Admin Users
Manage games, categories, and reviews (CRUD operations).
Access to an admin dashboard with statistics.

---

## Technologies Used
**Backend:** Laravel 12, PHP 8.2
**Database:** MySQL
**Frontend:** Blade Templates, Bootstrap 5
**Authentication:** Laravel Breeze / Fortify (email login, registration)
**Version Control:** Git
